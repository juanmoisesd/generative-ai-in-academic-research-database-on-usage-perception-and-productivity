# Contributing to Generative AI in Academic Research Database

Thank you for your interest! 🤖

## Ways to Contribute

### 📊 Data Contributions
- Report outdated data via [bug report](../../issues/new?template=bug_report.md)
- Suggest new data sources or variables via [feature request](../../issues/new?template=feature_request.md)
- Contribute survey data or bibliometric analysis results

### 💻 Code Contributions
1. Fork the repository
2. Create a feature branch
3. Commit with descriptive messages
4. Open a Pull Request
